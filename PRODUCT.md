# Product

## Register

product

## Users

Professional developers who ship with AI daily and feel their execution fluency decaying — syntax recall, pattern speed, the ability to write known code without a prompt box. First user: Bhargav (frontend engineer, interview-prepping). Context of use: short focused sessions (15–45 min), evenings or early mornings, day-job editor closed, often the last thing before bed. The job to be done: "keep my hands sharp enough for interviews and 2am incidents."

## Product Purpose

CodeReps is a coding fluency gym: timed practice reps (implement a known pattern from memory, build a component from a description, write a generic type) in an editor with no autocomplete, no AI, no paste. Instant local evaluation, a fluency score that decays without practice, spaced-repetition warmups, and streaks. LeetCode tests whether you can *solve*; CodeReps tests whether you can still *execute* what you already understand. Success = the daily warmup becomes a habit and per-category "interview readiness" is backed by evidence.

## Brand Personality

Training hall, timing instrument, honest. Serious and precise, quietly motivating — the emotional register of a 6am gym session, not a gamified app. Celebrates effort truthfully (counts paste attempts resisted; shows the decay when you skip a week). Never cute, never confetti.

## Anti-references

- LeetCode's utilitarian clutter and ad-cramped panels.
- Duolingo-style mascot cuteness and guilt-tripping.
- Generic "dark developer SaaS": neon-terminal-green cosplay, purple gradients, glassmorphism.
- AI-slop tropes: gradient text, identical icon-card grids, uppercase tracked eyebrow labels, hero-metric stat rows.

## Design Principles

1. **The editor is the hero** — during a rep, all chrome recedes; nothing competes with the code.
2. **Time is a material** — every numeral (timer, pars, splits, scores, streaks) is mono, tabular, instrument-grade.
3. **Honest metrics** — every score ships with its evidence; decay is shown, not hidden.
4. **Familiar affordances, unfamiliar identity** — components a Linear/Raycast user trusts instantly, in a voice no dev tool has (athletic timing equipment).
5. **Motion conveys state only** — splits land, digits tick, a PR stamp settles; nothing decorative.

## Accessibility & Inclusion

WCAG AA floor (body text ≥4.5:1 on dark). Pass/fail never color-alone — icon + word always. Full keyboard operability (this is a typing product; a rep must be completable mouse-free). `prefers-reduced-motion` alternatives for every animation. Dark-only at MVP by deliberate scene choice, not default.
